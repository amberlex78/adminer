# Adminer 4.8.1

If you have installed `PHP` on your local computer, you can start the `PHP server` with `Adminer`

## Install

```sh
git clone git@github.com:amberlex78/adminer.git
cd adminer
```

## Start Adminer

```sh
make start
```

## Stop Adminer

```sh
make stop
```

## Help

```sh
make
```
